โจทย์ทดสอบความพร้อมสำหรับฝึกงาน และสหกิจศึกษา

จากระบบที่ส่งมาพี่สามารถใช้งานได้แล้วครับ เนื่องด้วยทางน้อง Request ฝึกงานเป็นสหกิจศึกษา ทางบริษัทต้องทดสอบศักยภาพของนักศึกษาให้ถึงเกณฑ์ที่คาดหวัง ดังนั้น ทางบริษัทมีแบบทดสอบเพิ่มเติมให้ ดังนี้

ใช้ API จากงานที่เคยทำนะครับ

สร้าง Git Repository ใหม่ ใช้ HTMl, CSS, JavsScript ดึงข้อมูลจาก weather api เดิมที่เคยทำนะครับ โดยมีเงื่อนไขดังนี้

- เลือกข้อมูลเฉพาะจังหวัดในประเทศไทยเท่านั้น
- สร้างหน้า 1 หน้าชื่อว่า "พื้นที่สีเขียว"
- หน้าที่สร้างใหม่ หัวข้อว่า ค้นหาจังหวัดใกล้เคียงที่สภาพอากาศดีที่สุด
- กดเข้าหน้า มีข้อมูล select option จังหวัดในประเทศไทยให้เลือก + เลือกวันที่ + เลือกเวลา (วันที่เป็น 7 วันล่วงหน้า เวลาทุก 3 ชม. ตาม weather api)
- หลังจากกดปุ่มค้นหา แสดงตารางรายชื่อจังหวัดใกล้เคียงที่มีสภาพอากาศดีที่สุด จากสภาพอากาศดีที่สุดไปแย่ที่สุด ตารางประกอบไปด้วยคอลัมม์ดังนี้ จังหวัด, อุณภูมิ, ความเร็วลม, เมฆ, ระยะทาง (ระยะทางเป็นระยะทางที่ห่างจากจังหวัดที่กรอกข้อมูลเข้าไป)
- ความสำคัญการเรียงลำดับ ควรเรียงสถานที่ใกล้ที่สุด และสภาพอากาศดีที่สุด ก่อน หรือตามความเหมาะสมให้ makesense มากที่สุด
- หลังจากค้นหาเสร็จ สามารถกดค้นหาใหม่ได้ เลือกข้อมูลใหม่ได้โดยหน้าเว็บต้องไม่รีเฟรช
- ตกแต่ง CSS ให้สวยงามหน้าเว็บดูทันสมัย

อากาศดีหมายถึง 
อุณภูมิระหว่าง 15-25 องศา
ความเร็วลม < 20
เมฆ < 50%
